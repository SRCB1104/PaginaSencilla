# Tienda de Perfumes

Página web estática para mostrar y vender perfumes.

## Uso de la página
- El usuario puede ver el catálogo de perfumes disponibles.
- Cada perfume muestra su precio y descuento.
- Al dar clic en Comprar por WhatsApp, se abre WhatsApp con un mensaje automático para pedir el perfume.
- El botón flotante de WhatsApp permite contactar directamente con la tienda.

## Tecnologías
HTML y CSS
